1. ¿Qué diferencia hay entre JDK, JRE y JVM? ¿Por qué instalamos el JDK?
    - Cada uno de ellos menciona su función según sus siglas, el JDK es el Java Development Kit, que es básicamente todo
    el kit completo para trabajar con java.
    Por otro lado, el JRE, Java Runtime Enviroment sirve para poder correr los códigos, tiene dentro los JVM y las librerias
    Por último el JVM es el Java Virtual Machine, que es la que lee el bytecode, osease las clases, como mis dos programas
    Tengo que elegir el JDK porque no solo se piensa que trabaje haciendo codigos crudos en java, sino que al tener el kit completo, podemos crear proyectos, similar a los proyectos en el visual studio
